Mailing list​

Mailing list is recommended for discussing almost anything related to Nacos. Please refer to this?guide?for detailed documentation on how to subscribe to our mailing lists.

  • dev-nacos@googlegroups.com: The develop mailing list. You can ask questions here if you encounter any problem when using or developing Nacos.
  • commits-nacos@googlegroups.com: All commits will be sent to this mailing list. You can subscribe to it if you are interested in Nacos' development.
  • users-nacos@googlegroups.com: All Github?issue?updates and?pull request?updates will be sent to this mailing list.
  • nacos_dev@linux.alibaba.com.
